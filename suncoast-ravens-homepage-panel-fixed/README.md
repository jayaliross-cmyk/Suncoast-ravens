# Suncoast Ravens Website

This package uses the approved homepage mockup as the exact desktop homepage design.

## GitHub Pages
1. Unzip this package.
2. Upload the **contents** of the folder—not the ZIP itself—to the root of your GitHub repository.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/ (root)`.
6. The included `CNAME` file is already set to `suncoastravens.org`.

## Squarespace Domains
Your domain can remain registered with Squarespace Domains while GitHub Pages hosts the website for free. You will only update the DNS records in Squarespace Domains.

## Stripe
The sponsor buttons contain placeholders. Replace each placeholder link in `sponsors.html` with the corresponding Stripe Payment Link.
Do not place Stripe secret keys in this repository.

## Forms
The registration and contact forms are visual and functional in the browser, but they need a form endpoint such as Formspree, Google Forms, Basin, or a similar service to deliver submissions.
